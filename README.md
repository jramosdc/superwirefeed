# Superwire
Package of Cargobase UIKit and components showcase. Examples [here](https://cargobase.github.io/cargobase-ui-kit/).

## Contribution
To run a project, you probably need [Node.js](https://nodejs.org/en/download/) and [NPM](https://docs.npmjs.com/cli/install) installed.

---

### Install

To install dependencies you need to run:

    npm i webpack webpack-dev-server -g && npm i

To run, you can use:

    npm start

---

### Build

To build, please run:

    npm run build

Webpack will create a `dist` folder where you'll find all neccessary files which are required to run project.
